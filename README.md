# 项目简介

基于PyQt的深度学习行人重识别系统，主要用于视频监控中的行人识别和追踪。该系统功能模块包括数据预处理、深度学习模型训练、行人特征提取和匹配。技术特点方面，系统采用PyQt框架构建用户界面，确保操作便捷；结合深度学习技术，实现高准确度的行人重识别；支持多线程处理，提高系统运行效率。该系统不涉及售卖、交易等服务，专注于技术研究和应用。


## 核心功能
- 数据预处理：对监控视频数据进行清洗和格式转换，为后续分析做准备
- 深度学习模型训练：利用深度学习算法训练行人识别模型
- 行人特征提取：从视频帧中提取行人的关键特征信息
- 行人匹配：基于提取的特征进行行人身份的匹配和追踪
- 多线程处理：支持并行处理任务，提高系统运行效率

## 适用场景
- 视频监控系统中的行人识别
- 安防领域的人员追踪
- 智能监控系统开发
- 深度学习技术研究与应用

## 技术特点
- 采用PyQt框架构建用户界面，提供便捷的操作体验
- 结合深度学习技术，实现高准确度的行人重识别
- 支持多线程处理，提高系统运行效率
